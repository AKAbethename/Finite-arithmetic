# Калькулятор "большой" конечной арифметики

С помощью данного калькулятора можно производить действия над числами из множества {a, b, c, d, e, f, g, h}. На множестве задан линейный порядок a -- b -- d -- c -- h -- g -- e -- f.

Возможны следующие действия над числами: сложение, вычитание, умножение и деление с остатком.
